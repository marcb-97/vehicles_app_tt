# vehicles_app_tt
repositorio para proyecto
## Aplicación de anuncios de venta de coches
Aplicación web para explorar visualizaciones de un conjunto de datos de anuncios de venta de coches. Desarrollada con Streamlit y desplegada con Render
## Funcionalidades
Histograma: Distribución de kilometrajes
Diagrama de dispersión: Relación entre el año del modelo y el precio
